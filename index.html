내 핸드 (AP를 소모해 카드 사용)

나 HP 0 / 0

카드를 선택하면 타겟 숫자와의 수학적 관계 분석이 표시됩니다.

게임 규칙

  • 매 턴 3 AP가 주어지며, AP를 소모해 카드를 사용합니다. 일반 카드 1 AP, 스킬 카드 0~2 AP.
  • 🔗 수학 연쇄(Chain): 이전에 낸 카드 숫자와 약수/배수 관계인 카드를 내면 AP 0으로 무료 사용!
  • 한 턴에 최대 5장까지 사용 가능. 크리티컬 시 AP +1 회복 & 카드 한도 +1.
  • 연쇄 횟수만큼 콤보 보너스(+10%씩 중첩)가 적용됩니다.
  • 매 턴 적이 1~2개의 타겟 숫자를 제시합니다.
  • [크리티컬] N = GCD 또는 LCM → 3배 + 기절
  • [강화] N이 모든 타겟의 공약수/공배수 → 2배 + 방어막
  • [일반] 하나라도 약수/배수 → 1.5배
  • [기본] 관계 없음 → 1배
  • 적 처치 시 3장 중 1장을 선택해 덱에 추가. 덱 → 손패 → 버린 더미 순환.
